MATINS is a new 3D numerical code for magnetothermal evolution in neutron stars, based on a finite-volume scheme that employs the cubed-sphere system of coordinates.

Download

The latest version of MATINS is (WILL SOON BE!) available at:
https://example.com/matins-latest
