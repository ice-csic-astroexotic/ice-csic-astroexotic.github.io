MATINS is a new 3D numerical code for the MAgneto-Thermal evolution of Isolated Neutron Stars. It solves the TOV structure using a realistic, zero-temperature nuclear equation of state across the crust and core. The code also calculates temperature-dependent microphysical properties in both regions, which are essential for the coupled thermal and magnetic evolution. MATINS employs a finite-volume scheme built on the cubed-sphere coordinate system, representing the sphere with six connected faces like a cube and avoiding the axis singularities inherent to standard finite-volume methods.
